# Geometric Library - Документация

## Общее описание библиотеки

Библиотека geometric_lib содержит функции для подсчета площади и периметра геометрических фигур.

- **circle.py** - функции для вычисления площади и периметра окружности
- **square.py** - функции для вычисления площади и периметра квадрата
- **rectangle.py** - функции для вычисления площади и периметра прямоугольника
- **triangle.py** - функции для вычисления площади и периметра треугольникасшsqut

### square.py
**Формула:** S = a²<br>
**Параметр: a** - длина стороны квадрата (положительное число)<br>
**Возвращает:** площадь квадрата
```python
def area(a):
    return a * a

area(2)  # возвращает: 4
def perimeter(a)
```
**Формула:** P = 4 × a<br>
**Параметр:** a - длина стороны квадрата (положительное число)<br>
**Возвращает:** периметр квадрата<br>
```python
def perimeter(a):
    return 4 * a

perimeter(3)  # возвращает: 12
```

### triangle.py
**Формула:** S = (a × h) / 2<br>
**Параметры:** a – длина основания, h – высота треугольника (положительные числа)<br>
**Возвращает:** площадь треугольника

```python
def area(a, h):
    return a * h / 2

area(6, 4)  # возвращает: 12
def perimeter(a, b, c)
```

**Формула:** P = a + b + c<br>
**Параметры:** a, b, c – длины сторон треугольника (положительные числа)<br>
**Возвращает:** периметр треугольника

